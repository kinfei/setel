# Order Management Portal

Frontend ui of setel interview test to consume api(s) from backend service.

Functions(s):

- View list of orders
- Create new order
- View order detail
- Cancel order
- Polling to retrieve latest orders list

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start

```

```bash
http://localhost:3002
```
